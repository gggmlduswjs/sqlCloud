학습목표

1. inner 조인을 수행하여  여러 테이블로 부터 데이터를 조회할 수 있다 . (복습진행)
2. rollup 사용하는 경험을 갖는다  ( group by 선행되어야 한다)
3. sum()  over()   : 누적합을 구할 수 있다
4. 순위구하기 함수를 이용하여 상황에 맞는 순위를 구할 수 있다
   -rank() over( order by 컬럼명)
   -dense_rank() over (order by 컬럼명)
   -row_number() over( order by 컬럼명)
5. LAG() over(), LEAD()  over()  함수를 사용해 보는 경험을 갖는다 
6. ratio_to_report()  over() 함수 사용하여 비율을 구할 수 있다
6. decode함수를 사용하여 부서별 직급별 인원 수를 구할 수 있다
7. pviot, unpivot을 사용하는 경험을 갖는다 

  



 ########
 sum() over() 사용하여 누적합 구하기
 ########

--전체 누적값 구하기
 SELECT  p_total,  sum(p_total) over(  )
 FROM panmae;


--첫행에서 현재행까지의 누적 구하기
 SELECT  p_total,  sum(p_total) over( order by p_total)  
 FROM panmae;

--
 SELECT  p_total,  sum(p_total) over( order by p_total
    rows between unbounded preceding and current row   )
 FROM panmae;


--  unbounded preceding  (제한없는 이전의 뜻 : 처음부터라고 해석하면 됨)

 --첫행에서 현재행까지의 누적 구하기
 --기본값은 아래와 같다, 생략할 수 있다
 --같은 값은  같은  range로 묶인다
--필요하면  rows로 변경해야 한다
 SELECT  p_total,  sum(p_total) over( order by p_total
    range between unbounded preceding and current row   )
 FROM panmae;

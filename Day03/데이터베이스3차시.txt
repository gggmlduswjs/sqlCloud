용어정리
EQUI  조인 (동등조인) :   조인의 조건이  equal인   ( = )  조인
INNER JOIN (내부 조인): 두 테이블에서 조건이 일치하는 행들만 결과로 조회됨

용어설명:
equi : ON절에서 같은값을 비교하는 것을 의미
inner: 양쪽 모두에 있는것만 조회한다는 의미
 
JOIN 실습
 
 equi 조인 : 등가조인  ( 두 테이블 간에 같은 것이 있는것끼리 조인함 )
 ( 공통된 값이 있는 데이터만 반환함,   ON 절에서 같은 값을 비교 )

 

select * 
from 테이블명 A
join  테이블명  B
on  A.id = B.userId;
 

조인시 on절 조심하기 
 ( 두 테이블에 같은 항목을 갖고 있는것이 있다. 
두 개의 테이블은 한쪽에서 다른 테이블의 주키를 가지고 있음으로(즉 외래키)
두 테이블간의 관계가 설정된다)

 -inner join  ( inner 생략가능)
 -outer join   
    -left  outer join
    -right outer join
    -full outer join
